如果 build_exe.bat 沒有成功
==============================

必要條件
--------
1. 請先把 ZIP 完整解壓縮。
2. 不要直接在壓縮檔預覽畫面中執行 BAT。
3. 同一資料夾必須至少有：
   app.py
   demo.sql
   build_exe.bat

正確操作
--------
1. 雙擊 build_exe.bat。
2. 第一次安裝套件可能需要幾分鐘。
3. 成功後檔案會在：

   dist\DemoSQLQueryTool.exe

失敗時
------
同一資料夾會產生：

build_log.txt

請打開後，將最下面約 20 行貼給 ChatGPT。

手動建置指令
------------
也可以在該資料夾空白處按住 Shift，再按滑鼠右鍵，
選擇「在這裡開啟 PowerShell 視窗」，依序執行：

py -3.12 -m pip install --upgrade pip
py -3.12 -m pip install openpyxl pyinstaller
py -3.12 -m PyInstaller --noconfirm --clean --onefile --windowed --name DemoSQLQueryTool --collect-all openpyxl --add-data "demo.sql;." app.py

完成後 EXE 位於：

dist\DemoSQLQueryTool.exe
